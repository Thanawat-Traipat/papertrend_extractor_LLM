# โครงร่างงานวิจัย (Project Pitch)

## EIL Paper Analysis: การวิเคราะห์แนวโน้มหัวข้องานวิจัยในหลักสูตร EIL จุฬาลงกรณ์มหาวิทยาลัย

---

### ที่มาและปัญหาของการวิจัย

หลักสูตร English as an International Language (EIL) จุฬาลงกรณ์มหาวิทยาลัย เป็นหลักสูตรที่มีความหลากหลายทั้งในเชิงแนวคิดและผู้เรียน อย่างไรก็ตาม ยังไม่มีการศึกษาอย่างเป็นระบบเพื่อทำความเข้าใจลักษณะและแนวโน้ม (trend) ของงานวิจัยที่ผลิตจากหลักสูตร ทั้งในด้านประเด็นที่ผู้เรียนใน 3 tracks ได้แก่ **Linguistics, Language Assessment** และ **Language Instruction** ให้ความสนใจ รวมถึงการมุ่งเน้นด้านการสอนภาษา การเข้าใจภาษา หรือประเด็นอื่น ตลอดจนรูปแบบหรืออคติ (bias) ในเชิงภาษา ประเภทงาน และบริบททางวัฒนธรรม **ปัญหาหลัก**คือ เรายังไม่สามารถอธิบายภาพรวมและพัฒนาการขององค์ความรู้ที่ผลิตจากหลักสูตร EIL ได้อย่างเป็นระบบ

### คำถามวิจัย

1. มีแนวโน้มใดที่โดดเด่นในงานวิจัยของนักศึกษา EIL หรือไม่ และในแต่ละ track ผู้เรียนนิยมศึกษาประเด็นใดเป็นพิเศษ
2. งานวิจัยเขียนด้วยภาษาใด และศึกษาข้อมูลจากภาษาเดียวหรือหลายภาษา
3. มีอคติต่อประเภทงานวิจัย วิธีวิจัย หรือกลุ่มภาษาใดเป็นพิเศษหรือไม่
4. แนวโน้มของหัวข้องานวิจัยเปลี่ยนแปลงไปตามช่วงเวลาหรือไม่ และเชื่อมโยงกับปัจจัยภายนอกได้หรือไม่ เช่น การเพิ่มขึ้นของนักศึกษาต่างชาติ การเปลี่ยนแปลงนโยบายการศึกษา หรือกระแสทางภาษาศาสตร์ระดับสากล
5. จากข้อมูลที่ผ่านมา สามารถคาดการณ์แนวโน้มงานวิจัย EIL ในอีก 5–10 ปีข้างหน้าได้หรือไม่

### ข้อมูลที่ใช้

บทความวิจัยจากวารสารและการประชุมวิชาการ (Journal/Conference Papers) ตั้งแต่ **ปีการศึกษา 2549 จนถึงปัจจุบัน**

### แผนการดำเนินงาน

| ขั้นตอน | รายละเอียด |
|---|---|
| **1. Corpus Construction** | รวบรวมบทความทั้งหมดในช่วงเวลาที่กำหนด จัดหมวดหมู่ตาม track, ปีที่เผยแพร่ และประเภทงาน |
| **2. Descriptive & Trend Analysis** | สร้าง Timeline Dashboard แสดงวิวัฒนาการของหัวข้อวิจัยตามเวลา วิเคราะห์การเปลี่ยนแปลงในแต่ละช่วง |
| **3. Keyword & Topic Analysis** | วิเคราะห์ keyword ที่พบบ่อย และทำ Topic Modeling เพื่อหา theme หลัก |
| **4. Objective & Method Analysis** | วิเคราะห์คำกริยาที่ใช้ใน objective, ประเภทผลลัพธ์ (output) และวิธีวิจัย |
| **5. Author Background Analysis** | วิเคราะห์ความสัมพันธ์ระหว่างเชื้อชาติ/พื้นหลังผู้เขียน กับประเด็นที่เลือกศึกษา ตรวจสอบว่าผู้เขียนจากชาติใดมักศึกษาบริบทของชาตินั้นหรือไม่ |
| **6. Relationship Analysis** | วิเคราะห์ความสัมพันธ์ระหว่างตัวแปร เช่น track × topic, author background × research focus, time × methodology |

### ระเบียบวิธีวิจัย — ภาพรวม Workflow

```
Journal Papers
  → LLM-based Agent Extractor     ดึงข้อมูลสำคัญ (keywords, objectives, methodology, research focus)
  → Intermediate Structured Data   จัดเก็บในรูปแบบเชิงโครงสร้าง (tables / JSON)
  → LLM-based Merger & Analyzer   รวม วิเคราะห์ และหา pattern/trend จากข้อมูลทั้งหมด
  → Results Presentation           แสดงผลเชิงวิเคราะห์และกราฟแนวโน้ม
        • Textual Analysis
        • Visualizations (Graphs / Dashboards)
```

**เครื่องมือหลัก:** LangChain + LangGraph (LLM-based Multi-Agent Pipeline)
